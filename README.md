## 🚀 Getting Started

When you launch Burrow for the first time, you will see a clean workspace. This interface contains several key areas:

*   Side Panel: This area shows your file tree. You can create, rename, and delete folders or files here.
*   Terminal: The black area at the bottom of the window holds your command shell. This is where you run scripts and interact with the system.
*   Editor: The empty space in the middle is where you write your code. It supports syntax highlighting to help you read your work.
*   AI Assistant: A small icon in the top right corner triggers the local artificial intelligence. Use this to ask questions about your code or to generate snippets.

To start a new project, click the "New Folder" icon in the side panel. Name your folder and begin creating files within it. Burrow automatically saves your changes as you type.

## ⚙️ Using the Built-in Tools

Burrow comes with essential tools pre-installed. These tools allow you to perform common tasks without manual setup.

### The Shell
The shell acts as your command center. You can type instructions to navigate your file system or run programs. Because Burrow includes the Bun runtime, you can execute JavaScript and TypeScript files instantly without installing additional software suites.

### Git Integration
Burrow handles version control through its interface. You can link your project to a remote storage host. Click the "Source Control" tab on the left to see which files you changed. Provide a message in the box and click "Commit" to save your progress locally. You can push these changes to your external remote at any time.

### Local AI
The artificial intelligence feature runs on your local machine. It does not send your code to a cloud service. Type your request in the prompt box, such as "How do I create a function to add two numbers?" and the system will provide an explanation. It works offline once the initial model loads into your browser's local storage.
